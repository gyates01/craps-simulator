# Craps Simulator

A full-featured browser-based craps game with animated dice, drag-to-bet chips, and all standard bets.

## Features

- Animated dice rolling with physics
- Drag-and-drop chip betting
- Full craps ruleset (pass/don't pass, come/don't come, place, hardways, props)
- Session stats tracking

## Dev

```bash
npm install
npm run dev
```

## Tech

React + Vite + Tailwind CSS
